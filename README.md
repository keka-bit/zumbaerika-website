## Why this version is stable

This is a static site: `index.html` + `styles.css`.
There is no npm install and no Next.js build step, so Vercel should deploy it reliably.

## Deploy with Vercel

1. Create a GitHub repository.
2. Upload these files to the repository.
3. In Vercel, choose **Add New Project**.
4. Import the GitHub repository.
5. Framework preset: **Other** or **No Framework**.
6. Build command: leave blank.
7. Output directory: leave blank or use `.`.
8. Deploy.

## Updating photos later

Add photos to the `assets` folder, then update `index.html` to reference them.
